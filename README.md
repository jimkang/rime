rime
====

rime is a module that finds rhymes, including loose rhymes.

Installation
------------

    npm install rime

Usage
-----

Load the module.

    var rime = require('rime');

Get a loose rhyme.

    console.log(rime.getRhymesForWord('pock'));

Tests
-----

Run tests with `make test`.
